DV Landing Page Workshop

Build your landing page in 1 hour

Start

1. Let's setup

  1. Start Sublime Text. It will open a new black window
  2. Create a folder landing on your Desktop. Drag & drop this folder in the Sublime Text window
  3. In Sublime Text left navigation:
    • Right click "New File"
    • Save it as index.html with Cmd + S or Ctrl + S
    • Do the same to create a style.css file
    • Then "New Folder" and create an images folder
  4. Finally double click on index.html to open it with Chrome

2. Add some content

  • To save time, right-click on 4 of the following images and save them in your images folder. These will be your product feature icons, so choose wisely!
  • Then start with this HTML code in index.html
            <html>
  <head>
    <meta charset="utf-8">
    <title>My Venutre Landing Page</title>
  </head>

  <body>
    <h1>My New Venture</h1>
    <p>This is a very important introductory line!</p>
    <a href="">Explore</a>

    <img src="images/briefcase.svg" width="100">
    <h2>Serious</h2>
    <p>A serious app, <strong>very serious</strong> app</p>

    <img src="images/diamond.svg" width="100">
    <h2>Elegant</h2>
    <p>An elegant app, <strong>very elegant</strong> app</p>

    <img src="images/chat.svg" width="100">
    <h2>Social</h2>
    <p>A social app, <strong>very social</strong> app</p>

    <img src="images/spaceship.svg" width="100">
    <h2>Fast</h2>
    <p>A fast app, <strong>very fast</strong> app</p>

    <p>This is the footer of my landing page built at BCGDV</p>
  </body>
</html>

3. Basic CSS styling

  • Link your stylesheet style.css in the head section of your HTML
            <head>
  <link href='style.css' rel='stylesheet'>
</head>
  • Then copy/paste the CSS code below in style.css and change it around
            /* style.css */
body {
  background: #F5F5F5;
  color: green;
  font-size: 25px;
  font-family: Helvetica;
}
h1 {
  color: red;
  font-weight: lighter;
  font-size: 10px;
}
h2 {
  color: pink;
  font-weight: lighter;
  font-size: 8px;
}
a {
  color: yellow;
}
a:hover {
  text-decoration: none;
  color: purple;
}

5. Advanced CSS styling

  • Name your banner and footer with id like <div id="banner"> and <div id="footer">
  • Name your features with class like <div class="card">
  • Then add some nice CSS that uses these class and id
            /* style.css */
#banner {
  text-align: center;
  background-image: url("https://picsum.photos/1300/600/?random");
  background-size: cover;
  padding: 150px;
  color: white;
  text-shadow: 1px 1px 5px black;
}
#banner h1 {
  color: white;
  font-size: 50px;
}
#banner p {
  color: white;
  opacity: 0.8;
  font-size: 30px;
  font-weight: lighter;
}
#footer {
  padding: 30px;
  background: rgb(30, 30, 30);
  color: lightgrey;
}
.card {
  padding: 50px;
  font-weight: lighter;
}

6. Intro to Bootstrap

  • Link Bootstrap stylesheet in the head section of your HTML before your stylesheet link
            <head>
  <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css">
  <link rel="stylesheet" href="style.css">
</head>
